About This Project

This is a multi-threaded HTTP server implemented using low-level socket programming in Python.

  • Handles concurrent clients with a thread pool.
  • Supports GET for HTML, PNG, JPEG, and TXT files.
  • Supports POST for JSON data upload.
  • Implements security: path traversal prevention & host validation.
  • Supports HTTP/1.0 and HTTP/1.1 keep-alive connections.
